Google Sheet Online Check-In Functionalities

Functionalities

User Interface:

  • Custom menu creation for various actions (sending emails, creating contracts, generating codes, converting documents).
  • Alerts for unacknowledged messages tailored to the logged-in user.

Email Functionality:

  • Send personalized emails to users with contract details and attached PDFs.
  • Automatic logging of email timestamps in the Google Sheet.
  • Condition-based email sending based on user interaction (tick status).

Document Management:

  • Create new Google Docs from a template with user-specific data populated.
  • Store links to generated contracts in the Google Sheet.
  • Conversion of Google Docs to PDFs and logging of URLs.

Data Handling:

  • Retrieve and process data from specified columns in the Google Sheet.
  • Validate user interactions and update the acknowledgment status of messages.
  • Search functionality to find specific row indices based on column values.

Dynamic Content Creation:

  • Template-based document generation for contracts using user information (e.g., name, dates, addresses).
  • Formatting of dates and other data fields in the document.

Error Handling:

  • Logging of errors encountered during document creation or email sending for debugging purposes.
  • Skip rows in data processing based on predefined conditions (e.g., if certain columns are empty or already processed).

Random Code Generation:

  • Functionality to generate random door codes for rooms and update the corresponding cells in the Google Sheet.

User Notification:

  • Alerts to inform users about new messages or status updates within the system.
